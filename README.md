K-Means Clustering: This project applies K-Means clustering to group customers based on similarities in features such as annual income, spending score, and age. After preprocessing and scaling the dataset, the optimal number of clusters (K) is determined using the Elbow Method. The algorithm then assigns each data point to a cluster by minimizing within-cluster variance (inertia). The model is evaluated using inertia and silhouette score, and analysis is performed to understand cluster compactness, separation, and sensitivity to initialization. Customer segments such as high-income–high-spending groups are identified. Visualizations like elbow curves, scatter plots, and centroid plots help interpret clustering results.

GMM Clustering: Gaussian Mixture Models (GMM) are used to cluster data by assigning probabilistic membership to each point, enabling soft clustering. After preprocessing and scaling, the model is trained using the Expectation-Maximization algorithm with an optimal number of components. GMM captures complex, overlapping, and elliptical cluster shapes more effectively than K-Means. It provides both cluster labels and probabilities, allowing deeper insights into data distribution. Performance is evaluated using Log-Likelihood, AIC, BIC, and Silhouette Score. Analysis includes comparing GMM with K-Means, understanding cluster overlap, and examining flexibility. Visualizations such as contour plots and probability distributions help interpret clustering behavior
